OpenHours makes it easy to display your office opening/closing hours and your office's status in real time.

== Description ==

OpenHours makes it easy to display your office(s) opening/closing hours and your current status in real time.
It adds easily customizable and independant widgets to your Wordpress website. Simply drag one or more widgets in your theme where you want to display your opening hours.
Check a <a href="https://www.upsource.be/scripts/openhours-wordpress/">live demo here</a>




Have a look at OpenHours main features:   

#### Universal time format:

* 12 or 24 hour time format
* single or multiple time slots (Pro version)
* settings for adjusting timezones
* multiple display styles

#### 3 display modes:

* Dynamic Open/Closed text: displays the html text of your choice when your office is open or closed
* Opening hours timetable (full): lists every weekdays weither your office is open or closed
* Opening hours timetable (light): list only the days when your office is open

#### Easy CSS styling

* Generated source code is clean and powered with CSS class for futher custom styling
* Highlights the current day or not with provided active class

#### Page builders compatible

OpenHours widgets can be used in any page buiders plugins like Elementor.

#### Translation-ready

OpenHours is coded so you can easily translate both the frontend and backend.


== Installation ==

Follow the usual routine;

1. Open WordPress admin, go to Plugins, click Add New
2. Enter "openhours" in search and hit Enter
3. Plugin will show up as the first on the list, click "Install Now"
4. Activate & start using one or more Openhours Widgets

Or if needed, upload manually;

1. Download the latest stable version
2. Unzip it and upload to _/wp-content/plugins/_
3. Open WordPress admin - Plugins and click "Activate"
4. Start using one or more Openhours Widgets

Read the full documentation <a href="https://www.upsource.be/scripts/openhours-wordpress/documentation">available here</a> to learn how to setup your OpenHours widgets.
